# 能力管理板块 PRD 补充规格（仅本期范围）

## 1. 需求范围
1. 本期只交付能力管理板块，不包含智能体管理板块。
2. 能力管理包含：分类管理、能力列表、能力新增/编辑/查看/删除、问学同步能力只读展示。
3. 能力类型范围：问学应用、问学工作流、外部应用、外部工作流。

## 2. 术语定义
1. 同步能力：从问学平台同步而来，默认只读。
2. 自建能力：在本系统创建，仅支持外部应用、外部工作流。
3. 分类：能力归属分组，用于筛选和展示。

## 3. 数据模型（建议）
| 字段 | 类型 | 规则 |
|---|---|---|
| capability_id | string | 全局唯一，UUID |
| capability_name | string | 必填，去前后空格，最大 10 字符，分类内唯一 |
| capability_type | enum | 问学应用/问学工作流/外部应用/外部工作流 |
| category_id | string | 必填 |
| source | enum | sync/manual |
| intro | string | 选填，最大 10 字符 |
| request_config | json | source=manual 时必填 |
| created_by | string | 系统自动 |
| created_at | datetime | 系统自动 |
| updated_by | string | 系统自动 |
| updated_at | datetime | 系统自动 |
| is_deleted | bool | 软删除标记 |

## 4. 权限矩阵
| 操作 | 管理员 | 运营 | 只读 |
|---|---|---|---|
| 查看分类/能力 | 是 | 是 | 是 |
| 新增分类 | 是 | 是 | 否 |
| 编辑分类 | 是 | 是 | 否 |
| 删除分类 | 是 | 否 | 否 |
| 新增能力 | 是 | 是 | 否 |
| 编辑自建能力 | 是 | 是 | 否 |
| 编辑同步能力 | 否 | 否 | 否 |
| 删除自建能力 | 是 | 是 | 否 |
| 删除同步能力 | 否 | 否 | 否 |

## 5. 同步规则
1. 每 5 分钟增量同步一次，凌晨 02:00 全量校准一次。
2. 同步主键为 external_id + capability_type。
3. 冲突策略：同步字段覆盖本地同名字段；本地只读字段不允许人工改写。
4. 问学侧删除后，本地标记 is_deleted=true 并隐藏。
5. 单次失败重试 3 次，指数退避 1s/2s/4s。
6. 同步审计记录任务 ID、成功数、失败数、失败原因。

## 6. 分类管理规则
1. 分类名必填，最大 10 字符，去前后空格，不可重复。
2. 重复判定：忽略大小写，连续空格折叠为 1 个空格。
3. 系统内置分类不可编辑不可删除。
4. 删除分类时若分类下有能力，阻断删除并提示“请先迁移或删除能力”。
5. 分类排序规则：sort asc, created_at desc。

## 7. 能力管理规则
1. 新增能力仅允许外部应用、外部工作流。
2. 同步能力仅支持查看，不支持编辑和删除。
3. 能力名必填，最大 10 字符，分类内唯一。
4. 修改能力类型会清空 request_config，需二次确认。
5. 删除能力采用软删除，不物理删除。
6. 列表支持名称模糊、类型筛选、分类筛选。
7. 模糊搜索不区分大小写，使用包含匹配。

## 8. HTTP 请求配置规则
1. 支持 GET/POST/HEAD/PATCH/PUT/DELETE。
2. 配置项：URL、Header、Query、Body、超时。
3. 默认超时：连接 3s，读取 10s，写入 10s。
4. 超时范围：1-60s。
5. URL 必须为 http/https，禁止 localhost、回环地址、内网地址。
6. 禁止配置 Host、Content-Length Header。
7. 响应大小上限 2MB，超限失败。
8. 成功判定：HTTP 2xx 为成功，其余为失败。
9. 鉴权密钥加密存储，列表和日志脱敏展示。

## 9. 并发与一致性
1. 编辑接口使用乐观锁（updated_at 或 version）。
2. 冲突返回 409 + CAP_4091。
3. 删除和编辑接口必须幂等。

## 10. 错误码
| 错误码 | 场景 | 文案 |
|---|---|---|
| CAP_4001 | 参数校验失败 | 参数不合法 |
| CAP_4002 | 名称重复 | 名称已存在 |
| CAP_4003 | 分类不存在 | 分类已失效，请刷新 |
| CAP_4004 | 同步能力只读 | 同步能力不支持此操作 |
| CAP_4005 | 分类非空不可删 | 分类下存在能力，无法删除 |
| CAP_4006 | URL 不允许 | URL 不在允许范围 |
| CAP_4091 | 并发冲突 | 数据已变更，请刷新后重试 |
| CAP_5001 | 外部调用失败 | 外部服务异常，请稍后重试 |

## 11. 审计与日志
1. 记录新增/编辑/删除：操作者、时间、变更前后值。
2. 记录外部请求：能力 ID、trace_id、耗时、状态码。
3. token、密钥、cookie 必须脱敏。

## 12. 非功能要求
1. 1 万条能力数据下，列表查询 P95 < 500ms。
2. 同步任务具备可观测性（成功率、延迟、失败原因）。
3. 服务可用性目标 99.9%。
